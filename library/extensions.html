Toggle Text

M I N I - P L U G I N

A simple plugin that adds the ability to dynamically toggle text by adding a single data atrribute and value on an element.

DIV'S TOO!
& Links, Too!

PLEASE NOTE:

When using this mini-plugin on anchor tags ( <a> ), the plugin calls the JavaScript method Event.preventDefault(), so a user can see the anchor text swap. This also prevents any page scrolling, so you can use anchor elements as interactive UI elements along with the text toggle functionality.

There are also mouseover & mouseout listeners added to each initialized element to change the cursor to a pointer upon a user :hover interaction.

And finally, the script converts specific characters ( &, <, >, ", and ' ) to HTML entity values during the swaps to prevent any issues when toggling text content using those characters.
